# URL-Shortener-
A tool that serves to shorten your website links to bitly.
# How to Install
```
$ sudo apt-get install python
$ git clone https://github.com/hidayat-code/URL-Shortener-
$ cd URL-Shortener-
$ python UrlShortener.py 
```
